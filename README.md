# Paladin2

基于Spring Boot、NG-ZORRO(Ant Design of Angular)的后台管理基础框架

> 立项时间：2019-1-7


# 说明

本项目分离了前后端，共分为2个主工程

- 前端工程：`paladin-boss`

- 后端工程：`paladin-api`

## paladin-boss
- 核心：Angular `7.0.0`
- UI：NG-ZORRO(Ant Design of Angular)

## paladin-api

- 核心：Spring Boot `2.1.1`
- 持久层：Mybatis Plus `3.0.7.1`
- 权限：Shiro `1.4.0`
